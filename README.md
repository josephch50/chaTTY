# chaTTY

**A way to access TTY sessions over a web broswer**

## Pre-requisites
- [Node.js](https://nodejs.org/en/download/)

## Installation
```bash
# Clone this repository
git clone https://github.com/josephchen50/chaTTY
# Go into the repository
cd chaTTY
# Install dependencies
npm install
# Run the app
npm start
```
